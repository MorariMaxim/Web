Principalele Functionalitati

1. Autentificare si Profil

Utilizatorii isi vor putea crea un profil si loga pe el

Cu fiecare profil sunt asociate imagini descarcate de pe platforme sau incarcate de pe computerul utilizatorului

2. Descarcarea de imagini de pe anumite platforme

Platformele disponibile:

  • Imgur
  • ...

Aceste imagini sunt din 2 categorii:

  • Imagini personale de pe platforma
  • Imagini cautate dupa numite criterii, nu obligatoriu apartinand utilizatorului pe platforma

La dorinta se pot descarca si metadata legate de imagini:

  • Comentarii
  • Evaluări
  • Etichete
  • Număr de partajări

Aceasta functionalitate poate fi accesata prin pagina "Platform Download" din popout menu de pe headerul aplicatiei

Utilizatorul poate alege pentru cautare platforma, keywords, rating minim, etc.

Efectul cautarii va fi redirectionarea la o pagina ce afiseaza rezultatele cautarii

Imaginile dorite pot fi selectate si asociate efectiv cu profilul utilizatorului

3. Photo Upload

Se pot uploada imagini de pe computerul utilizatorului

Pentru asta a se vedea pagina "Local Upload" din popout menu

Imaginile pot fi selectate prin drag and drop sau cu ajutorul file system managerului, e.g. "File Explorer"

4. Vizualizarea de imagini

Imaginile asociate unui profil pot fi vizualizate convenabil

Pagina de vizualizare se numeste "Gallery" si se gaseste in popout menu(coltul din stanga sus)

Imaginile vizualizate se pot filtra dupa criterii asemanatoare cu cele de pe pagina "Platform Download"

Imaginile pot fi selectate si se poate efectua operatii asupra lor, spre exemplu "upload," "download," "delete," etc.

5. Photo Focus

Oricare imagine se poate vizualiza izolata de altele, in contrast cu pagina "Gallery"

Pentru imaginea data se vor mai si afisa comentarii, numarul de likuri asociate, etc. , daca este cazul

Pentru a accesa aceasta pagina, se apasa butonul "Focus" din meniul de actiuni din pagina "Gallery"

Tot de pe aceasta pagina, apasand butonul "Edit", se poate accesa urmatorul grup functional

6. Editarea Imaginilor

Imaginile se pot edita in cateva moduri

  • Aplicarea de filtre precum "sepia," "invert," "contrast," "grayscale"
  • Adnotarea unei parti a imaginii cu taguri

La fel se pot decupa parti ale imaginii cu ajutorul unui dreptunghi mobil deasupra ei

7. API

Aplicatia ofera un API pentru pentru accesarea imaginilor prin intermediul unui serviciu REST

API se gasestea la /API

Pe ramura /API/get?id=IMAGE_ID se poate descarca o anumita imagine indicand id-ul asociat ei

Id-urile se pot primi de pe ramura /API/search?... indicand anumite criterii:

  • keywords=KEYWORD1,...KEYWORDN
  • minRating=RATING
  • platform=PLATFORM
  • altele ...
