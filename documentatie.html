Principalele Funcționalități

1. Autentificare și Profil

Utilizatorii își vor putea crea un profil și loga pe el.

Cu fiecare profil sunt asociate imagini descărcate de pe platforme sau încărcate de pe computerul utilizatorului.

Un utilizator nelogat apasă pe "Log In" din header și este redirecționat la pagina de login/signup.

Acolo el poate crea un nou profil sau se poate loga pe unul existent introducând un username și password.

2. Descărcarea de imagini de pe anumite platforme

Platformele disponibile:

  • Imgur
  • ...

Aceste imagini sunt din 2 categorii:

  • Imagini personale de pe platformă
  • Imagini căutate după anumite criterii, nu obligatoriu aparținând utilizatorului pe platformă.

La dorință se pot descărca și metadata legate de imagini:

  • Comentarii
  • Evaluări
  • Etichete
  • Număr de partajări

Această funcționalitate poate fi accesată prin pagina "Platform Download" din popout menu de pe headerul aplicației.

Utilizatorul poate alege pentru căutare platforma, keywords, rating minim, etc.

Efectul căutării va fi redirecționarea la o pagină ce afișează rezultatele căutării.

Imaginile dorite pot fi selectate și asociate efectiv cu profilul utilizatorului.

3. Photo Upload

Se pot uploada imagini de pe computerul utilizatorului.

Pentru asta a se vedea pagina "Local Upload" din popout menu.

Imaginile pot fi selectate prin drag and drop sau cu ajutorul file system managerului, e.g. "File Explorer".

4. Vizualizarea de imagini

Imaginile asociate unui profil pot fi vizualizate convenabil.

Pagina de vizualizare se numește "Gallery" și se găsește în popout menu (colțul din stânga sus).

Imaginile vizualizate se pot filtra după criterii asemănătoare cu cele de pe pagina "Platform Download".

Imaginile pot fi selectate și se pot efectua operații asupra lor, spre exemplu "upload," "download," "delete," etc.

5. Photo Focus

Oricare imagine se poate vizualiza izolată de altele, în contrast cu pagina "Gallery".

Pentru imaginea dată se vor afișa comentarii, numărul de like-uri asociate, etc., dacă este cazul.

Pentru a accesa această pagină, se apasă butonul "Focus" din meniul de acțiuni din pagina "Gallery".

Tot de pe această pagină, apăsând butonul "Edit", se poate accesa următorul grup funcțional.

6. Editarea Imaginilor

Imaginile se pot edita în câteva moduri:

  • Aplicarea de filtre precum "sepia," "invert," "contrast," "grayscale".
  • Adnotarea unei părți a imaginii cu taguri.

La fel se pot decupa părți ale imaginii cu ajutorul unui dreptunghi mobil deasupra ei.

7. API

Aplicația oferă un API pentru accesarea imaginilor prin intermediul unui serviciu REST.

API se găsește la /API.

Pe ramura /API/images/{id} se pot afla date despre o anumită imagine.

Pe ramura /API/getImage?id={id} se poate prelua imaginea în sine (jpeg, png ... file).

Id-urile se pot primi de pe ramura /API/images... indicând anumite criterii:

  • tags={KEYWORD1},...{KEYWORDN}
  • origin=(uploaded|imgur|unsplash|strong)
  • title={in_title_string}

ex: /api/images?origin=edit&tags=desert,dunes&title=my first edited image
